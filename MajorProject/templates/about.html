An assistance system for visually challenged using Machine Learning

About the Project

This work focuses on emotion recognition. Emotion reveals valuable information regarding human communications. It is common to use facial expressions to express emotions during a conversation and some interpersonal communication can be achieved using facial expressions only. If a machine were able to interpret its user’s facial expression correctly, it might be able to help its user more efficiently. In this work, an emotion recognition system is introduced for the visually impaired people using machine learning. Machine learning is one of the current technologies that use computers to perform tasks like humans. Inspired by the advances in Convolutional Neural Networks (CNNs) achieved in image recognition and classification, we propose an approach to address this problem. The experimental results show that after adopting this method, overall robustness and accuracy of facial expression recognition have been greatly improved compared to those obtained by transfer learning.

Facial expression which is used to represent various emotion’s is the most effective form of non-verbal communication, and it provides intimation about emotional state, mindset, and intention.

Entire project is subdivided into the following steps five steps that’s Image acquisition, face detection, feature extraction, emotion recognition or classification we have used FER2013 dataset which consists of 35,685 examples of 48x48 pixel grayscale images of faces.  

Methodology Adopted

  1. Data Pre-processing:- First step was to select appropriate dataset so for our project work we opted for FER 2013 dataset.

  2. Model training:- We used pre-processing techniques including resizing, reshaping, and grayscale conversion. 

  3. Implementation:- We used two approaches one is CNN based and other transfer learning using FER dataset.

  4. Text to Speech Conversion:- In this stage our model will be used for real time emotion identification from web camera of laptop desktop and the emotion, which is classified will be expressed as an audio through audio device. 

Features of our Project

  1. It can be helpful for the better interaction among the blind and normal person
  2. Blind can be more open to other person if knowing the expression
  3. Dont need to rely on 3rd person for considering emotion expression of other
  4. Our project aims for better development of the blind among society
  5. Individuals with autism frequently fail to react appropriately to the emotions of others, and researchers have suggested that an impairment in emotional expression recognition may contribute to these inappropriate reactions and to their understanding of emotion in general.
